Маппинг товаров

Как связать новую позицию: один раз «Обновить фид» и «Обновить кэш МС». Слева «Показать фид» — только строки без маппинга. Нажмите «Взять key → искать в МС» у нужной позиции: подставится key и справа появятся кандидаты из МойСклада. Выберите строку «Выбрать», при необходимости поправьте поля внизу и нажмите «Сохранить маппинг». Полностью без рук нельзя — в МС и в фиде разные идентификаторы, вы подтверждаете связку.
Товары из фида
«Показать фид» — непромапленные позиции из фида. «Обновить фид» — скачать свежий YML с сайта.
Кэш ассортимента МойСклад
«Пусто» здесь не значит «нет товаров в МС» — локальный кэш не загружен или не совпадён поиск. Сначала «Обновить кэш МС», затем «Искать МС» (пустой поиск — первые 200 позиций).

Рассылка в боте

При включенной тестовой отправке массовая рассылка не выполняется: сообщение уходит только админу.
При выключенной тестовой отправке сообщение отправляется всем пользователям.
