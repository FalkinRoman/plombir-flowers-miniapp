Про ссылку «В МС» и поля связки
Ссылка «В МС ↗» в правой колонке: если в API пришло meta.uuidHref, открываем его (как в JSON МойСклада) — так веб чаще ведёт на нужную карточку. Если uuidHref нет — строим URL из meta.href / id. В поля связки по-прежнему идут ms href и id с конца Remap API (meta.href), не uuidHref — заказы в МС создаются по API-ссылке. Достаточно MOYSKLAD_TOKEN на сервере; после «Обновить кэш МС» подтянется uuidHref в кэш.
