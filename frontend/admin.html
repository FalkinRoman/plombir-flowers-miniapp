Загрузка счётчиков…
Идите по кнопкам сверху по порядку: подсвечивается текущий шаг (чёрная кнопка), выполненные — зелёные.
Поле справа от кнопок — не отдельный «поиск по сайту», а фильтр строк слева перед шагом 3: вводишь кусок названия/key → «Показать непромапленные» покажет только подходящие позиции из фида.
Tilda — витрина
Слева позиции из фида (каталог сайта / YML). Здесь берём key варианта товара.

Выбери строку

Сначала шаги 1→3 сверху, затем выбери товар.
МойСклад
Справа кандидаты из локального кэша номенклатуры МС (не напрямую из облака при каждом клике).

Кандидаты (авто после выбора слева)

Быстрый поиск по кэшу: артикул, код, ext, название. Несколько слов через пробел — все должны встретиться (как фильтр в справочнике).
Выбери товар слева — справа топ-8 с оценкой. Пусто? Обнови кэш МС (шаг 2).
Связка для мини-приложения и заказов
Эти поля — то, что сохраняется в базе и используется бэкендом при оформлении заказа из Telegram mini app: по tilda key подставляется позиция МойСклада. Без сохранённой пары заказ может создать «левую» номенклатуру.
Расширенные действия

Рассылка в боте

При включенной тестовой отправке массовая рассылка не выполняется: сообщение уходит только админу.
При выключенной тестовой отправке сообщение отправляется всем пользователям.
